# Banking API

API stateful simples que simula sistemas bancários internos para desenvolvimento
local e demonstração do agente.

## Rotas

- `POST /v1/customers`
- `GET /v1/customers/{customer_id}/profile`
- `PUT /v1/customers/{customer_id}/profile`
- `GET /v1/customers/{customer_id}/balance`
- `PUT /v1/customers/{customer_id}/balance`
- `GET /v1/customers/{customer_id}/card-limit`
- `PUT /v1/customers/{customer_id}/card-limit`
- `POST /v1/pix`
